# installation
1. download files(the dmg)
2. put file in a folder expample: "terminal game"
3. then cd into the directory "terminal game"
4. type sudo cp -f .build/release/TG /usr/local/bin/TG
5. type cd again
6. en type TG
7. and play!

# remove
1. type cd /usr/local/bin/
2. type sudo rm TG
3. there you go!
